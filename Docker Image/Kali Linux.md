## Kali Linux

* Pull
```bash
$ docker pull kalilinux/kali-linux-docker
```

* Run
```bash
$ docker run -t -i kalilinux/kali-linux-docker /bin/bash
```

* Install Metasploit Framework
```bash
$ apt-get update && apt-get install metasploit-framework
```
